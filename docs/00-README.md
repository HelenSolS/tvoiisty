# ТЗ-пакет: «Твой ИИ-стиль» (MVP)

Этот пакет — единый источник правды для разработки MVP виртуальной примерочной «Твой ИИ-стиль» (Telegram Mini App + backend + БД + AI-воркер). 

## Цель MVP
Доказать, что 3 источника изображений (фото пользователя + образ магазина + локация админа) **стабильно** собираются в результат:
1) Пользователь загружает фото.
2) Выбирает образ из каталога.
3) Опционально выбирает локацию (10 штук доступны всем на старте).
4) Нажимает «Примерить» → получает готовую примерку.
5) Сразу доступно: скачать, поделиться, анимировать, перейти в магазин.

## Главное правило MVP
Никаких усложнений «двух контуров» в реализации MVP. Архитектура должна **позволять** два контура позже, но MVP должен работать на текущем доступном стеке.

## Роли
- **User** — обычный пользователь.
- **Store** — пользователь, который включил режим магазина (в MVP: заглушка подтверждения через Госуслуги/Т-Банк).
- **Admin** — администратор платформы (управляет локациями, модерацией, фичефлагами).

## Этапы
- Этап 1 (MVP): фото пользователя + образ + локация → генерация фото → архив/лайки → переход в магазин → (опция) анимация → (опция) постинг.
- Этап 2+: стилист (для магазина и для пользователя), расширенные локации, продвижение, платежи токенов, реальные верификации.
