# 02. UX/UI спецификация (минимализм + темы)

## 2.1. Тональность и визуальный стиль
- Минимализм, “ничего не мешает”.
- Мягкие формы, тонкие линии, воздух.
- 3 темы (переключаемые пользователем):
  - Мята
  - Персик
  - Лаванда
- Темы меняют: акцентный цвет, подсветку кнопок/иконок, выделение “ИИ” в названии.

## 2.2. Логотип и написание
Название: **«Твой ИИ-стиль»**
- «Твой стиль» — обычным шрифтом.
- Буквы **«ИИ»** — чуть выше по базовой линии и окрашены акцентом выбранной темы.

## 2.3. Навигация (MVP)
Нижняя навигация (3–4 пункта):
- Примерка
- Образы
- Локации
- Профиль

Кнопка “В магазин” видима:
- в карточке образа
- в результате примерки

## 2.4. Экран «Примерка» (главный)
Одна задача — быстро собрать 3 компонента и нажать «Примерить».

Блоки сверху вниз:
1) **Моё фото**
   - пустое состояние: “Добавьте фото” + подсказка требований.
   - если фото загружено: превью + кнопка “Заменить”.

2) **Образ**
   - если не выбран: кнопка “Выбрать образ” → открывает каталог.
   - если выбран: карточка образа (превью) + “Изменить”.

3) **Локация (опционально)**
   - тумблер “Нейтральный фон / Локация”.
   - если включено: горизонтальная лента из 10 локаций (превью).

4) CTA
   - кнопка **«Примерить»**.
   - при нажатии без авторизации → экран авторизации → возврат к процессу.

## 2.5. Каталог «Образы»
- Вверху фильтры (минимум): пол + тип.
- Карточка образа:
  - превью
  - бейджи (пол/тип/сезон)
  - название/магазин
  - кнопка “Примерить”
  - кнопка “В магазин”

Важно: каталог должен поддерживать бесконечную подгрузку/пагинацию (чтобы не повторилась проблема “видно 4 и дальше нельзя”).

## 2.6. Экран «Локации»
MVP: просто 10 локаций.
- Карточки превью.
- При выборе — возвращаемся в «Примерка», выбранная локация подставлена.

## 2.7. Экран «Результат»
Показывает готовую примерку.
Кнопки:
- Скачать
- Поделиться (Telegram share)
- Анимировать (если включен этап 2)
- Повторить (retry)
- В магазин

Состояния:
- Генерация: прогресс + статус (“Проверяем фото…”, “Готовим примерку…”, “Финальный рендер…”)
- Ошибка: причина + кнопка “Повторить”

## 2.8. Экран «Профиль»
- Аватар, имя (из Telegram)
- Переключатель темы: Мята / Персик / Лаванда
- Раздел “Мои примерки” (история)
- Кнопка “Стать магазином”

Для магазина:
- Статус: Не подтвержден / Подтвержден
- Кнопка “Загрузить образ”
- Раздел “Статистика”

## 2.9. Требования к загрузке изображений (UI)
Перед загрузкой пользователь видит правила:
- формат: JPG/PNG
- минимальная сторона: 1024px (желательно)
- вес: до 10–15MB
- лицо/тело: допускается селфи и полный рост
- запрет: порнография (эротика допустима только на уровне приличия, но “порно — нет”)

Если фото не подходит:
- понятная ошибка (“слишком маленькое”, “слишком размытое”, “не прошло проверку приличности”).
