# Канон: репозитории и домены

**Это единственный источник правды по именам. Не путать.**

---

## tvoiisty (этот репозиторий)

| Что | Значение |
|-----|----------|
| **Имя** | **tvoiisty** (одно слово, на «y») |
| **GitHub** | https://github.com/HelenSolS/tvoiisty |
| **Содержимое** | Backend (API, примерка, БД, движок) + текущий фронтенд |
| **Деплой backend** | api.tvoiistyle.top (VPS) |
| **Деплой фронта** | tvoiisty.vercel.app (текущий интерфейс) |
| **Роль** | Основное приложение, источник API и логики |

---

## tvoiistyle (другой репозиторий)

| Что | Значение |
|-----|----------|
| **Имя** | **tvoiistyle** (на «yle», = «стиль») |
| **GitHub** | https://github.com/HelenSolS/tvoiistyle |
| **Содержимое** | Только фронтенд. Второй интерфейс (дизайн под инвестора). |
| **API** | Обращается к api.tvoiistyle.top (backend из tvoiisty) |
| **Деплой** | Vercel → tvoiistyle.vercel.app, затем app.tvoiistyle.top |
| **Роль** | Новый UI. Логику и движок не дублирует. |

---

## Правило

- **tvoiisty** — здесь живут backend и текущий фронт. Один репо, полный стек.
- **tvoiistyle** — только новый фронт, другой репо. Движок и API остаются в tvoiisty.

При работе в этом репо (tvoiisty) не подменять его именем tvoiistyle и не переносить сюда код из tvoiistyle без явной договорённости.
