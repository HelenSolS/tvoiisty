

```bash
cd /opt
rm -rf newstyle
git clone https://HelenSolS:ТВОЙ_ТОКЕН@github.com/HelenSolS/newstyle.git
cd newstyle/newstyle
docker compose build --no-cache
docker compose up -d
curl -s http://localhost:4001/health
```

Токен потом смени в настройках GitHub и заново сделай `git pull` с новым токеном.

---

## Если что-то пошло не так

- **«docker: command not found»** — на сервере не установлен Docker. Напиши, установим.
- **«Permission denied»** — перед шагом 8 выполни: `chmod +x deploy.sh`
- **Архив не найден** — убедись, что шаг 4 выполнен с Mac и архив действительно залился: на сервере выполни `ls /root/newstyle-deploy.tar.gz`
- **tsc: not found при сборке** — сделай `git pull origin main` в папке с проектом и снова `docker compose build --no-cache`
