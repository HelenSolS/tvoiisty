# основное правило

Frontend, который утверждён инвестором, нельзя переделывать.

Backend должен подстроиться под него.

# запреты

нельзя:

- переписывать UI
- менять структуру экранов
- делать прямые вызовы AI из frontend
- использовать base64 pipeline.

# требования

все операции генерации должны проходить через:

- queue system.

хранение медиа

все изображения должны сохраняться через storage layer.

ошибки генерации

всегда должен существовать fallback provider.
